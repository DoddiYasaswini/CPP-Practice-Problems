## 1. Even or Odd

### Problem Statement
Determine whether a given number is even or odd.

### Algorithm
1. Read the number.
2. Check n % 2.
3. If remainder is 0 → Even.
4. Else → Odd.

### Dry Run

Input:
8

Execution:
8 % 2 = 0

Output:
8 is even number

### Time Complexity
O(1)

### Space Complexity
O(1)
